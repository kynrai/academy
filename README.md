# Academy for Developers

An intensive 11-week course for developers covering the full stack from terminal basics to agentic AI.

## Building the book

Install [mdBook](https://rust-lang.github.io/mdBook/guide/installation.html), then:

```sh
# Serve locally with live reload
mdbook serve

# Build static output to book/
mdbook build
```
